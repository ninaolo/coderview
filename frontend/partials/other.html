Pitfalls


Highest/lowest product problems

Highest/lowest product doesn't only involve highest/lowest numbers. Also think of negative numbers --> two very large negative numbers multiplied = a large positive. Easy to forget!


Finding a sum among integers

Not needed to combine all with all (N^2). Previously seen Integers can be stored in HashMap and the remaining needed number(s) expressed with the sum.


Recursion

Don't forget the base case! Otherwise, you'll end up with infinite recursion.


Bit operations

ALWAYS put parenthesis around bit operations. Example: if ((a & b) > 0) {} --> otherwise compile error.

Bit manipulation


AND

Multiply numbers --> 0 * 1 = 0 and so on...

            
                int a = 60; // 00111100
                int b = 13; // 00001101
                int checker = 8; // 00001000
                int insert = 254; // 11111110

                // Result: a AND b = 00001100 = 12.
                int c = a & b;

                // Use case 1: check a certain bit.
                if ((b & checker) > 0) {
                    System.out.println("Bit nr 3 is 1.");
                } else {
                    System.out.println("Bit nr 3 is 0.");
                }

                // Use case 2: insert 0 somewhere.
                int d = b & insert; // 00001100 = 12

OR

Result = 1 if first OR second is 1.

            
                int a = 60; // 00111100
                int b = 13; // 00001101
                int insert = 1; // 00000001

                // Result: a OR b = 00111101 = 61.
                int c = a | b;

                // Use case: insert 1 somewhere.
                int d = a | insert; // 00111101 = 61

XOR

Exclusive OR. Result = 1 if both are exclusive (10 or 01).

            
                int a = 60; // 00111100
                int b = 13; // 00001101

                // Result: a XOR b = 00110001 = 49.
                int c = a ^ b;

NOT

Inverse the binary number. IMPORTANT: last bit = sign bit becomes 1 --> number becomes negative.

            
                int a = 60; // 00111100

                // Result: NOT a = 1...11000011 = -61.
                int c = ~a;

SHIFT LEFT/RIGHT

Shifts bits n steps to left/right.

            
                int a = 60; // 00111100
                int b = 13; // 00001101

                int c = a << 2; // 11110000 = 240
                int d = a >> 2; // 00001111 = 15
